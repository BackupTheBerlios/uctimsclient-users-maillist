Hi Lenia

Have you tried opening the streams with VLC? Are the correct rtsp urls
placed in the configuration file? Try copying the rtsp address from the
config file and see if VLC can open it. 

You might want to also try use Wireshark on both the client machine and
the server running Darwin to determine what the SIP signalling is doing
as well as to see if any media is being streamed and what ports it is
being streamed to if it is.

Regards
Robert
