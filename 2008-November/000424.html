Hi,

UCT IMS Client v.12 is Ubuntu Hardy compatible, which uses libvlc0 
0.8.6e / libvlc-dev 0.8.6e.  If you install this version of the package 
you will be able to compile.

I suspect you have a later version of the libvlc libraries, where the 
API has been modified.  We are working on v.13 that is Ubuntu Intrepid 
compatible, and works with libvlc2 0.9.4 / libvlc-dev 0.9.4.

Check which version of libvlc you are running, then you can either 
install the older version or I can send you an SVN version of the new 
client (Still some warnings/bugs!).

Regards,
Richard.
