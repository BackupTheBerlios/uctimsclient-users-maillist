Hi All,

This is just a quick update on the status of the UCT IMS Client and 
related projects.

The latest version, v 1.0.12 of the client is now available for 
download.  As well as general bug fixes and improved standards 
conformance the client now incorporates MSRP (Session Based IM) support, 
RTSP (Media Control) support and basic Electronic Program Guide (EPG) 
support.

Additionally the UCT IPTv has developed a new release v 1.0.3.  This 
project involves an Indirection Application Server (AS) and relevant 
RTSP enabled media server.  Essentially the client initiates a session 
(Can use the EPG interface) with the Indirection AS; the AS maps this 
request to a specific RTSP address and returns this to the client - an 
RTSP session is setup between the client and the relevant media server.

Further information for installation/configuration can be found on the 
website http://uctimsclient.berlios.de, as always please send any bugs, 
feature requests or comments to uctimsclient-users at lists.berlios.de

Regards,
Richard.
