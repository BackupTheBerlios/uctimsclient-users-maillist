> Hi,

> Based on the initial filter criteria (iFC) set up in the HSS the S-CSCF
> forwards SIP request onto relevant Application Servers.  In the case of
> the UCT IPTv Server the Initial Filter Criteria in the HSS are set such
> that any INVITE request with the phrase "iptv" in the To SIP Header is
> forwarded to the UCT IPTv Server, here the request is further analysed
> and the UE is added to the broadcast list of the requested channel.
