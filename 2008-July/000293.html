  Hi,

Based on the initial filter criteria (iFC) set up in the HSS the S-CSCFforwards SIP request onto relevant Application Servers.  In the case ofthe UCT IPTv Server the Initial Filter Criteria in the HSS are set suchthat any INVITE request with the phrase 'iptv' in the To SIP Header isforwarded to the UCT IPTv Server, here the request is further analysedand the UE is added to the broadcast list of the requested channel.
