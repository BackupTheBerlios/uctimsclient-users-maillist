Hi Khaldon
 
Two questions:
 
1. Can the second machine that the client is on use the DNS, i.e. can
it ping pcsf.open-ims.test, as apposed to the IP address?
 
2. Do you have a default route setup in linux? I seem to recall that
eXosip complains (possibly with a Network Unreachable error) if you
don't have a default route, even if you don't need it. Try
route add default gw 192.168.1.7
 
Hope that helps.
