i have been usings repository data to store user's information from my
Aplication Server in the HSS. I use JavaDiameterPeer because I use a
Java Application Server, and also the Diameter commands PUR and UDR.

You can use JavaDiameterPeer in your AS to create and start up a peer,to
create a DiameterMessage and the AVPs you need to use in your Diameter
command, and to handle the sending and reception of diameter messages.
