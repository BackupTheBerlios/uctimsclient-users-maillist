Hi Pedro,

The installation guide is designed to work with Ubuntu, so there are a 
few differences between distributions.

A blank screen implies that Tomcat cannot find the web application - for 
Ubuntu the folder for webapps is /var/lib/tomcatX.X/webapps, while I 
think with Fedora the webapps directory is /usr/share/tomcatX.X/webapps. 
So this is where you should be copying the uct_web_pcm folder.

There could be other differences - e.g. the default listening port for 
non secure connections (For Ubuntu this is 8180, check 
/etc/tomcatX.X/server.xml for this), and the tomcat policy folder.

Regards,
Richard.
