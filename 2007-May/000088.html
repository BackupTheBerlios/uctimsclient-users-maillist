Hi Kiran,

The UCTIMSClient is primarily development software and uses several 
Linux libraries, so it does not run natively in Windows.   Alternatively 
you can use a Virtual Machine (like VMWare) or run cross platform 
software (like CrossOver Office) to run the client in Windows.

regards,

Richard
