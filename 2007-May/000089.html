Hi Kiran,

That config file has worked correctly with OpenSER v1.2.0 and OpenSER 
presence module v1.2.0 on a Debian architecture (see  
http://www.openser.org/pub/openser/1.2.0/packages/deb-stable/)  What 
version do you have running?  Are the presence module and OpenSER 
package the same version?  Also I noticed openSER is running on a 
different port, what other changes did you make to the config file?

Also have you run openser_mysql create to setup the neccessary presence 
MySQL tables correctly?
