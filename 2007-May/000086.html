Hi All,

The latest version (1.0.4) of the UCTIMSClient is now available for 
download at http://uctimsclient.berlios.de/

This version has an improved messaging facility, built-in presence 
functionality (see 
http://uctimsclient.berlios.de/presence_server_howto.html) and other 
minor fixes.
As well as the usual source tarball, a debian package is now available 
for easy installation of the UCTIMSClient on debian architectures.

As always this is a work in progress and we welcome all comments, bugs,
etc.
