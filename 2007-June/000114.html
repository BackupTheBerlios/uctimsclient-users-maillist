We had this same problem. It is a bug in the OpenIMSCore before release 
281 (see openIMScore archives from 29 May). Download and install the 
latest core and it should sort itself out.

There also seems to be a bug with subscribing to the reg event. If this 
fails then often the subscriptions to your buddies presence will also 
timeout. I will investigate this problem at a later date but I am almost 
certain it is a bug in the core.
