Hi Kedy

The client was designed from scratch to be an IMS client. The SIP mode 
was only introduced because at first we had nothing to test it against 
except other SIP clients (chicken and egg type thing).

Now that the OpenIMScore has taken off in popularity and a lot of people 
seem quite comfortable using it I'm considering scrapping SIP mode all 
together, since it hasn't been looked at in ages.

So in conclusion there is no SIP-only client. However, I can suggest 
linphone which is the SIP client I looked at when we started with our 
own client.
