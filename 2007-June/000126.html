Hi

This problem is occuring while linking (not compiling).

This problem is probably a result of a badly written makefile. Perhaps 
you have extracted this version into a directory with an older version 
of the client.

I suggest running

make clean

just to clear out any old objects fom the folder. Hope this helps, 
otherwise its a more serious problem.
