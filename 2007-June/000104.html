>>>> Hi Li,
>>>>
>>>> I think the MySQL modules comes standard with OpenSER_1.2.0 - check 
>>>> /usr/lib/openser/modules, and if mysql.so is present then the 
>>>> module is already loaded (If this file is not here then you must 
>>>> install the mysql module).
>>>>
>>>> When OpenSER is used as a presence server it uses its own MySQL 
>>>> database to store and retrieve the presence info. i.e. No 
>>>> information is taken from the HSS, all the IMS Core does is forward 
>>>> SUBSCRIBE and PUBLISH messages with event = presence to the 
>>>> presence server.
>>>>
>>>> The presence module makes new tables in the OpenSER database that 
>>>> allow for presence functionality.  Once you have installed OpenSER 
>>>> and the presence modules you should be able to run the script
>>>> openser_mysql create
>>>>
>>> >from the the command line.  This script adds the correct tables to the
>>>> OpenSER database.
>>>>
>>>> I am not sure which Linux distribution you are using, but:
>>>> mysql-client
>>>> mysql-server
>>>> libmysqlclient-dev
>>>> libxml
>>>>
>>>> are all standard packages in the debian repositories.
>>>>
>>>> regards,
>>>>
>>>> Richard
