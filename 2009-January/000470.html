Hi,

Version 1.0.13 of the UCT IMS Client is now available for download from 
the client website (http://uctimsclient.berlios.de).  As well as general 
bug fixes and standards compliance, this version compiles with libraries 
standard in the Ubuntu Intrepid repositories.  Particularly the client 
can now be built with libvlc0.9.x.

The latest version of Ubuntu has had some issues with v4l libraries, and 
there may still be some bugs with video and camera support.

Please post all bugs, comments and suggestions to 
uctimsclient-users at lists.berlios.de

Regards,
Richard.
