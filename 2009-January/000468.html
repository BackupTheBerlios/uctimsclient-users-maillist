As well as the DNS configuration you need to make sure that the Diameter 
configuration files are correct.  Specifically: pcscf.qos.xml,
DiameterPeerPCEF.xml and DiameterPeerPCRF.xml.  Examine these files and 
ensure that each Diameter Peer is correctly specified.
