Can you confirm that audio and video packets are flowing to the client?

The client uses the xvimagesink element to display the video. You can 
test if this element is working on your machine using this command:

gst-launch-0.10 videotestsrc ! xvimagesink

You should see a TV test pattern. If not gstreamer is not working 
correctly on your machine.

Regards

Dave
