As far as I recall the server just hangs at the line "elements linked" 
if there is an error.

Make sure if you are just using one file you specify one channel, i.e.

./uctiptv 500000 1 media/Dilbert101.avi

I have tried with many different types of files and most work (with the 
exception of real media). This is what the output looks like if it works:
