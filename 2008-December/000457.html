Hi Yang,

The configurator script in the ser_ims package and the ser_ims_qos
package is identical. Essentially all this script does is text
replacement, it search es for all "127.0.0.1" and replaces with your IP
address, and all "open-ims.test" and replaces with your domain. It can
be run on any file - when running the script pass as parameters the
files you wish to reconfigure.

e.g. cd /opt/OpenIMSCore/ser_ims/cfg
./configurator.sh /opt/OpenIMSCore/uctpcrf/config/*

Regards,
Richard.
