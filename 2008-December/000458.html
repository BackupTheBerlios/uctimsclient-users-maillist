Hi Yang,

With the separation of media and signaling the RTP packets follow the
shortest path between the UE's - not necessarily going through the PCEF.

To force the RTP traffic through the PCEF, you need to enable an RTP
proxy (see http://www.openimscore.org/node/76) and then run the PCEF on
that machine. This will force all RTP traffic through that machine. In
the case that you have an RTP proxy configured, you should use the PCSCF
config file - /pcscf.qos.rtp.cfg.

/Regards,
Richard.
