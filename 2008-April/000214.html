Hi Chitra

If those gstreamer commands do not work, then there is a problem with your
gstreamer and media handling, not necessarily with the IPTV server and ims
clients.

I have no idea why your gstreamer setup is not working, but if you get those
log files to me i will try to find out.

You can also try capture the network traffic with wireshark and see if the
media is actually being sent.

The IPTV window should show up as it is part of the application, and is
hidden / showned when appropriate - this is based on the session being
setup, not actually on the media flow.

If you send me a wireshark trace or the log files of those gstreamer
commands being run with GST_DEBUG=3 I will try find out what is going wrong.

Kind Regards
Richard
