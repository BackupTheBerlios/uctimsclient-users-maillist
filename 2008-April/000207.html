Hmm this is a rather more complex issue. From what I understand you are 
just looking to display the name of a movie in the display area from the 
server. In this case you may want to just add an extra attribute in the 
SDP message and when the SDP message is parsed set the display with it.

On a separate but somewhat related issue we did look at implementing a 
EPG for the server, the EPG was stored in xml on the XCAP server and the 
client could request the file and the client would then pop up a window 
display the programming information. It was near final but I think there 
was a problem with the information being displayed multiple times and I 
couldn't figure out how to clear some of the GTK widgets (the drop boxes 
especially IIRC) in order to stop this. However if there is a need for 
it we could look at implementing it in the next release of the client.

Robert
