HI Eduardo,

I use Debian OS and I have vlc installed from the package.
You don't need to do anything in the client.
You need to create proper trigger in the hss so that your INVITE requests
will be passed to IPTV server.
Then just run vlc:

/usr/bin/vlc -I http --http-host localhost:9280 --extraintf telnet
--telnet-host localhost:9212 --telnet-password **** --rtsp-host
0.0.0.0:9254--vlm-conf //home/ims/vlc/cfg/vlm.conf

the most important is --rtsp-host wich tells media stream address.
I use vlm config to configure available streams because it is simpler:

new Test vod enabled
setup Test input http://home.in.tum.de/~paula/mpeg/wg_gdo_2.mpg

and then just configure iptv server to point do this vlc stream source in my
case:

<key-value_pairs>
<key-value_pair>
        <key>channel1</key>
        <value>rtsp://some_ip:9254/Test</value>
</key-value_pair>
</key-value_pairs>

that's it:)

Cheers
Tomasz
