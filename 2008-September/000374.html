If you are using Ubuntu then the libvlc-dev library can be found in the 
package manager. I presume there are libvlc packages for most other 
linux distributions as well.

Don't worry about the other errors regarding mediacontrol, those are 
part of libvlc and will no longer be present when you install the -dev 
package.
