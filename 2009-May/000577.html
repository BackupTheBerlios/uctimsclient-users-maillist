> Anyone using Ubuntu Jaunty will have noticed the video encoding in the
> UCT IMS Client no longer works - this is because the H263 encoder is
> missing.  The fix for this is:
>
> sudo apt-get install libavcodec-unstripped-52
> sudo apt-get install ffmpeg
> rm -rf /home/user_name/.gstreamer-0.10/
>
> This makes Gstreamer rebuild the list of plugins.   Thanks to Richard
> Spiers for this fix.
