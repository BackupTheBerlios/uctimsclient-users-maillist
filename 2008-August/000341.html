Hi,

The latest related packages available in the Ubuntu hardy repositories are:
libexosip2-4 (v 3.0.3)
libosip2-2 (v 3.0.3)

UCT IMS Client v 1.0.10 and onwards works with these packages.

The old packages in previous Ubuntu repositories (gutsy, etc.) are:
libexosip2-5
libosip2-3

UCT IMS Client V1.0.9 and older versions work with these packages.


If you want to use v1.0.10 and onwards you either need to upgrade to 
Ubuntu Hardy to get access to the newer packages; temporarily update 
your repository to point to Hardy and get access to the newer packages; 
or directly download the relevant packages from the Ubuntu Hardy website.

Regards,
Richard.
