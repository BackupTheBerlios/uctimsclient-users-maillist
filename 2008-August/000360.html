Hi,

Is that the exact error message - or is it more like:

[Thread-4] ERROR de.fhg.fokus.diameter.DiameterPeer.peer.StateMachine  -
StateMachine: Peer pcrf.open-ims.test not responding to connection
attempt 

If this is the case and the error does not persist there is no problem.
This message will appear if the Diameter peer (This case the PCRF) is
not yet running.  Once you run the PCRF this error should not appear
again.

If this is not your problem - have you correctly modified the
DiameterPeerXXX.xml files in the config folder to change your domain
name and IP address?

Regards,
Richard.
