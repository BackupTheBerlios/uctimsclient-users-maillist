Hi Miroslav,

You seem to be confusing the UCT developed UCT IMS Client with the 
Fraunhofer FOKUS developed Open Source IMS Core.

In future please post any questions related to the UCT IMS Client to the 
uctimsclient users list - while those related to the Open Source IMS 
Core should be posted to the openimscore users list.

For adding an Application Server into the FHoSS of the Open IMS Core see 
the Provisioning Services section of the FAQ's on the Open IMS Core 
page: http://www.openimscore.org/faq
