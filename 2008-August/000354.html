>     Hi Khaldon,
>
>     The UCT IMS Client utilises the Gstreamer framework for media
>     transport.
>     For the media to sent and received correctly you should have the
>     following packages installed:
>
>     gstreamer0.10-plugins-base
>     gstreamer0.10-plugins-good
>     gstreamer0.10-plugins-bad
>     gstreamer0.10-plugins-ugly
>     gstreamer0.10-ffmpeg
>
>     Also you need to run the clients on separate machines for audio to
>     work correctly  as only one client can interface to the sound card
>     at a time.
>
>     Paste any errors reported in the UCT IMS Client terminal if you
>     still have problems.
>
