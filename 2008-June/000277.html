Hi Miroslave

At the moment neither the UCT IMS Client or IPtv server have any RTSP 
support, but we will look into adding this in future versions. If you 
are desperate I believe that FOKUS has a commercial IPtv solution that 
has this functionality plus a whole lot more. VLC also has support for RTSP.

If you want to implement this yourself I think SIP content indirection 
to an RTSP stream would be the best solution. although it's not clear to 
me if this fits in with the various IPtv standards - most of which seem 
to be lacking any useful signaling examples.

Regards

Dave
