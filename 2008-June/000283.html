Hi Pedro,

Please find attached a Wireshark library for the Gx interface - it will 
be included with the next software release.  It works fine - although my 
Wireshark (v 1.0.0) seems to be mixing up which Diameter messages are 
Requests and which are Answers.

Also note - the Gx interface in uctpcrf v.1.0.2 is based on 3GPP TS 
29.212 v7.0.0 (2007-03) and is part of release 7.  A later version of TS 
29.212 is available (v8.0.0 (2008-5)).  The next release of the software 
will incorporate these new AVPs and the Wireshark library will also be 
updated (The same will be done for the Rx interface).

Regards,
Richard.
