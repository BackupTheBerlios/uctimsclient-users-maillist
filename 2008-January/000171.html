Hi Abhinav

The problem with the ximagesink / xvimagesink is only on the client side 
(UCT IMS Client). This we will have to deal with later.

Your problem is with the server not starting correctly. This is often 
just a case of your video file not being compatible. Try with another 
file, preferably an avi. Also make sure you have the ffmpeg gstreamer 
plugins installed.

Regards

Dave
