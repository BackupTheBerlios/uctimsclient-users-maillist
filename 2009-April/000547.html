Hi

Create your resource list document in a text editor, set the correct XDMS
preferences (XCAP root, username and password) in the client and upload from
the XDMS tab using the appropriate AUID, i.e. resource-lists
Please note that if you use the Register as Alice / Bob button the XDMS
username and password preferences will also be changed.
