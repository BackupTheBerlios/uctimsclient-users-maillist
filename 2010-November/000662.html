> > There are two points here:
> > 1- As cristhian explained, you need to use ubuntu 8.4. I couldn't find
> > compatibility information in the official website, however, this is the only
> > way I got it working.
> > 2- The error message "DEBUG [get_if_output] connect : Network Unreachable"
> > at the indirection server suggests network problem or DNS problem.
> >
> > what I can suggest is to use a video client (e.g. VLC) at the
> > AS_Indirection server, or at the end-user PC, and see if you can stream the
> > video using the same url in the Indirection server configuration file. If it
> > works, then there is an issue in the Indirection server integration with the
> > IMS Core (which I doubt, given the error message). if it doesn't work, try
> > to check the network and the DNS by pinging the DNS-name and the IP address
> > of the streaming server and see which one responds.
> >
> > hope that helps
