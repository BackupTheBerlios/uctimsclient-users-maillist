Yes that will be possible. If you are using the zipped file version of the
client, you can modify the source code and re-compile the client.
Otherwise, if you are using the debian version of the client, then after
you have added the new user-profiles to the HSS, you can register as the
new user(s) by typing in their sip address in the URI entry on the client.
i.e sip:bob1 at domain . Then click "Register" under the options menu.
