  Hi Asma,

  

It sounds like you manually installed these packages and may not have 

put them in their default locations.  Wherever possible use package 

managers to install software as these use the default locations.

  

From the error message it seemd that the PKG_CONFIG_PATH environment 

variable needs to point to the folder containing the ortp package.  Try
  

a search for ortp.pc then set the environment variable to this location.

i.e. /export PKG_CONFIG_PATH="/rtp/package/location"

  

/regards,

  

Richard
