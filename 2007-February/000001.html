Those configure scripts and makefiles were put in by Glade. We must have 
forgotten to remove them. The only thing you should do to install is 
type "make" from the ./src directory and then ./uctimsclient

The list of required libraries can be found here: 
http://uctimsclient.berlios.de/documentation/

If that doesn't work you may need to change the makefile to suit your 
needs. It's very simple so don't worry.
