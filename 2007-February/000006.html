That would not be an easy task. To make our lives easy we went with 
linux, and for now there are no plans to port to windows.

The client uses GTK for the GUI, and several other linux-specific 
packages (alsa, etc.). I imagine it would probably be easier just to 
start from scratch, or modify an existing windows SIP client.

The FOKUS IMS client is java-based, but last time I checked it was 
not-free (probably very expensive).
