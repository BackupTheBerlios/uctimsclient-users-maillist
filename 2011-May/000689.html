Hi there,
I have noticed that many people experienced the segmentation fault
problem when they are trying to use UCT IMS Client with a recent
version of Ubuntu.
It seem that nobody provided yet the solution which is to update the
source code in order to use the newest functions of libvlc library.
I have modified myself the source code to make uctimsclient v1.0.13
compatible with libdev v1.1.4-1ubuntu1 which is the version that can
be installed from Ubuntu 10.10 (Maverick Meerkat) repository (it goes
with vlc 1.1.4 The luggage).

Four files need to be updated :
- rtsp.c
- globals.h
- extern_vars.h
- includes.h

If I provide them to the community, is there any chance that the UCT
IMS Client can be updated with them ?
