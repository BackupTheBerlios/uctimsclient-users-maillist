try compiling uct from source. You must first, install -dev version of
all requried packages (vlc, libosip exosip and so on) if you wont -
make will be complaining about lack of some header files. Install
those with apitiude (or other package manager on Ubuntu)

Download tar.gz, untar, enter the folder and type "make" - if you are
not familiar with compiling apps from source, search tutorial over the
net - for example http://luv.asn.au/overheads/compile.html

uctimsclient has only one simple Makefile, so you must only type
"make" and you'll have binary file named "uctimsclient".

Also remember to mail posts to uctimsclient-users at lists.berlios.de
also, so it'll go to the mailing list :)
