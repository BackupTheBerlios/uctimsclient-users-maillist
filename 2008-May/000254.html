Hi,

I haven't managed to replicate your errors - but I do get compilation 
errors with the uctiptv v.1.0.0 package - possibly there are some API 
changes, this version should be removed from the website.

Please check the version of your libexosip2 and libosip2 packages in 
your package manager -  if you are using v.3.0.3 use uctiptv v.1.0.2, if 
you are v2.2.3 use uctiptv v.1.0.1.

I don't think this will entirely solve your problem - but we should be 
able to find the problem faster with the later software versions.

Regards,
Richard.
