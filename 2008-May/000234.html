Hi

Are you calling between two UCT clients? If not, the client might not 
support the codec of the other client.

For calling between two UCT clients make sure that video calling is 
enabled in both the calling and receiving clients.

Only one client can access the camera at a time so if you want full 
duplex calling you will need to call between two different machines both 
with their own web-cams. It is also possible that your camera is not 
supported by gstreamer. You can test this from the terminal using the 
following command:

gst-launch-0.10 v4lsrc ! ffmpegcolorspace ! ximagesink

Regards

Dave
