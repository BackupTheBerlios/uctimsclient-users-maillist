Hi,

Unfortunately this option is not catered for in the compilation - I will 
rectify this in the next release.

For now we can hack it.  I have attached a modified configurator.sh 
script that will replace all /opt/OpenIMSCore/uctpcrf with the given 
installation location.

This script should be run on all config files, all JSP files, and all 
PCRF java files - so copy the script to the PCRF directory and:
./configurator config/*
./configurator src/pcrf/*
./configurator uct_web_pcm/pages/*
 
The script will prompt you for the new installation location - To cater 
for escape characters etc. your installation location should be entered 
as followed:
if your installation location is /home/ims/pcrf you should enter 
\\/home\\/ims\\/pcrf

hope this helps,

Regards,
Richard.
