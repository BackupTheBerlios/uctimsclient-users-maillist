Hi Miroslave,

Which version of libeXoSIP and liboSIP are you using? And which version 
of the UCT IPTv package are you using?

In the latest Ubuntu release the oSIP and eXoSIP stacks have been 
updated to version 3.0.3 and this has some slight API changes to the 
previous version (2.2.3).

UCT IPTv v.1.0.2 is compatible with v. 3.0.3 of libeXoSIP and liboSIP, 
while UCT IPTv v.10.1 is compatible with the older v. 2.2.3 of libeXoSIP 
and liboSIP.

Regards,
Richard.
