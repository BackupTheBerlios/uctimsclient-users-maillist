>
> I solved problem, so I'll write solution in case someone will need it.
>
> In file /etc/tomcat5.5/policy.d/03catalina.policy I added following  lines:
>
> grant codeBase "file:${catalina.base}/webapps/uct_web_pcm/-" {
>                        permission java.security.AllPermission;
> };
>
> Works fine now :)
>
> Regards,
> Emina
